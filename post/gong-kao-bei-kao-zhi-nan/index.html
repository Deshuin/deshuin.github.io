公考备考指南

2023-10-01 3 min read # 公考 # 考试 # 就业

一、行测

1. 行测题型

2. 行测作答策略

  1. 言语理解与表达、判断推理、资料分析，先做自己擅长的题型，顺序不唯一
  2. 预留5+分钟做数量，挑2一4道简单题，剩下的直接蒙，蒙未出现的选项
  3. 常识可以最后做，直接涂卡，不会的直接蒙，不犹豫

3. 行测备考核心

  1. 【第一类】片段阅读、文章阅读、逻辑判断、资料分析：建立知识框架，明确核心考点、系统学习
  2. 【第二类】图形推理、类比推理、定义判断：掌握常考规律，坚持刷题提升敏感度
  3. 【第三类】数量：建立知识框架，明确核心考点、系统学习
  4. 【第四类】常识判断、选词填空：积累知识，拓宽知识面和词语储备

4. 行测备考需要做的6件事情

  1. 读：最新政策方针、评论员文章、习近平讲话。
  2. 背：资料分析公式、高频成语。
  3. 算：基本计算能力、速算技巧。
  4. 学：基础知识、高频考点、解题方法
  5. 练：多维度刷题、针对性刷题，做题习惯，强优补弱。
  6. 模：限时实战、全真模拟、做题策略。

    • 熟读人民日报那些与习思想、”二十大”相关连载评论：
  2. 精做山东真题：《山东申论近三年题目》，反复做3遍；
  3. 精学五大题型：五大题型作题方法，每种题型精炼至少20道：
  4. 训练阅读敏感：每天至少阅读6000字文章材料：
  5. 积累实务方法：积累基层实务处理方法，学会20个经典问题处理案例：
  6. 掌握分析能力：掌握多角度分析能力，每天看新闻评论类文章至少一篇：
